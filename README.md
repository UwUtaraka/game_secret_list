# Список секретов игр
## Программа является списком чит кодов, багов и отсылок в разных играх, как если бы вы их записывали в блокнот.

## :exclamation:ЗАПУСК ТЕСТОВ:exclamation:
Для запуска тестов нужно иметь **CMake** и **CMake Tools**. Для сборки тестов, нужно открыть **test.cpp**, после чго вверху в VS Code в поиске прописать ```>CMake: Build```
![image](https://github.com/UwUtaraka/game_secret_list/blob/main/images/image.png)
После этого соберётся исполняемый файл **run_tests.exe**. Для запуска тестов, нужно после сборки из директории **build** в терминале прописать ```./run_tests.``` 
![image1](https://github.com/UwUtaraka/game_secret_list/blob/main/images/image1.png)

## Что умеет программа:
- Поиск игр по названию
- Добавление новых записей к уже существующим
- Добавление новой игры
- Удаление игр из списка

## Программа
- Язык: C++
- Библиотеки: [nlohmann/json](https://github.com/nlohmann/json) (для работы с JSON)
- Тестирование: Google Test

## Как работать с программой
## :exclamation:Перед закрытием программы, важно находиться в **главном меню**, чтобы не сломать базу данных, в противном случае, база обнулится :exclamation:
### Вывод всех игр:
Для вывода всех игр, нужно при запуске программы выбрать действие **[1]**, что выведет весь список игр

### Вывод игр конкретной категории:
Для вывода всех игр конкретной категории, нужно при запуске программы выбрать действие **[2]**, что выведет все игры одной категории

### Поиск игры по названию
Для поиска игры по названию, нужно при запуске программы выбрать действие **[3]**, что выведет искомую игру, если она имеется

### Добавить игру
Для добавления игры, нужно при запуске программы выбрать действие **[4]**, после чего выбрать категорию, после нужно будет ввести название игры:
- если игра уже есть, будет предложено:
  - Добавить новую информацию к данной категории
  - Перезаписать данные полностью
  - Отменить действие и вернуться назад
- если игры нет, будет предложено:
  - Написать чит код(желательно также написать, что он делает)
  - Написать название бага, если оно есть, а также описать его суть, после чего выставить примерный уровень сложности бага
  - Описать суть пасхалки, если выбрана категория пасхалок
  
### Удалить игру
Для удаления игры, нужно при запуске программы выбрать действие **[5]**, после чего будет предложено:
- окончательно выбрать удаление
  - Написать номер игры, которую нужно удалить из списка
- отменить действие(вернёт на главное меню)  

### Выйти
Для закрытия приложения можно закрыть просто окно, либо же в главном меню нажать **[6]**
